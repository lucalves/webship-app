## Install

```bash
# Clone this repository
$ git clone git@github.com:lucalves/webship-app.git

# Go into the repository
$ cd webship-react-app

# Open the repository in the IDE (If you use VS Code)
$ code .

# Install dependencies with npm or yarn
$ yarn or npm install
```
